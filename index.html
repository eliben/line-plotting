Line plotter

Plots 2D functions of y(x) in a given range.

y(x): the function to plot. Use JS syntax, assuming x is pre-defined. For example, Math.sin(Math.PI * x) will plot a sinusoid.

Num points: how many points to plot; the points are generated with linear spacing in the given (inclusive) range. The plot will use linear interpolation between points, so plots with a small number of points will be jagged approximations.

Start X: value of x to start the plot from.

End X: value of x to end the plot at.
